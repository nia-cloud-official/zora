Returns the full path of a unique filename that looks like\+: {\ttfamily dir/prefix-\/7ddd44c0} or {\ttfamily dir/7ddd44c0}

{\itshape dir} – The path you want the filename in. {\ttfamily os.\+tmpdir()} is a good choice for this.

{\itshape fileprefix} – A string to append prior to the unique part of the filename. The parameter is required if {\itshape uniqstr} is also passed in but is otherwise optional and can be {\ttfamily undefined}/{\ttfamily null}/`'\textquotesingle{}\`{}. If present and not empty then this string plus a hyphen are prepended to the unique part.

{\itshape uniqstr} – Optional, if not passed the unique part of the resulting filename will be random. If passed in it will be generated from this string in a reproducable way.
