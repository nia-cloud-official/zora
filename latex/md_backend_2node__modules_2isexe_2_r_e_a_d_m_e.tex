Check if the path is executable. If no callback provided, and a global {\ttfamily Promise} object is available, then a Promise will be returned.

Will raise whatever errors may be raised by {\ttfamily fs.\+stat}, unless {\ttfamily options.\+ignore\+Errors} is set to true.\hypertarget{md_backend_2node__modules_2isexe_2_r_e_a_d_m_e_autotoc_md1756}{}\doxysubsection{\texorpdfstring{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}{{\ttfamily isexe.\+sync(path, \mbox{[}options\mbox{]})}}}\label{md_backend_2node__modules_2isexe_2_r_e_a_d_m_e_autotoc_md1756}

\begin{DoxyItemize}
\item {\ttfamily ignore\+Errors} Treat all errors as "{}no, this is not executable"{}, but don\textquotesingle{}t raise them.

\item {\ttfamily gid} Number to use as the group id
\item {\ttfamily path\+Ext} List of path extensions to use instead of {\ttfamily PATHEXT} environment variable on Windows.
