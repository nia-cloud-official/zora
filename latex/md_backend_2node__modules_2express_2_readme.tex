\begin{DoxyCode}{0}

View the website at\+: \href{http://localhost:3000}{\texttt{ http\+://localhost\+:3000}}\hypertarget{md_backend_2node__modules_2express_2_readme_autotoc_md1230}{}\doxysubsection{\texorpdfstring{Philosophy}{Philosophy}}\label{md_backend_2node__modules_2express_2_readme_autotoc_md1230}
The Express philosophy is to provide small, robust tooling for HTTP servers, making it a great solution for single page applications, websites, hybrids, or public HTTP APIs.

Express does not force you to use any specific ORM or template engine. With support for over 14 template engines via \href{https://github.com/tj/consolidate.js}{\texttt{ Consolidate.\+js}}, you can quickly craft your perfect framework.\hypertarget{md_backend_2node__modules_2express_2_readme_autotoc_md1231}{}\doxysubsection{\texorpdfstring{Examples}{Examples}}\label{md_backend_2node__modules_2express_2_readme_autotoc_md1231}
