\hypertarget{md_backend_2node__modules_2simple-get_2_r_e_a_d_m_e_autotoc_md3306}{}\doxysection{\texorpdfstring{features}{features}}\label{md_backend_2node__modules_2simple-get_2_r_e_a_d_m_e_autotoc_md3306}
This module is the lightest possible wrapper on top of node.\+js {\ttfamily http}, but supporting these essential features\+:


\begin{DoxyItemize}
\item follows redirects
\item automatically handles gzip/deflate responses
\item supports HTTPS
\item supports specifying a timeout
\item supports convenience {\ttfamily url} key so there\textquotesingle{}s no need to use {\ttfamily url.\+parse} on the url when specifying options
\item composes well with npm packages for features like cookies, proxies, form data, \& OAuth
\end{DoxyItemize}

All this in \texorpdfstring{$<$}{<} 100 lines of code.\hypertarget{md_backend_2node__modules_2simple-get_2_r_e_a_d_m_e_autotoc_md3307}{}\doxysection{\texorpdfstring{install}{install}}\label{md_backend_2node__modules_2simple-get_2_r_e_a_d_m_e_autotoc_md3307}
