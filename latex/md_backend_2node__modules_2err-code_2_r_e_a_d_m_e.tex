If the passed {\ttfamily Error} already has a {\ttfamily .code} field, or fields specified in the third argument to {\ttfamily errcode} they will be overwritten, unless the fields are read only or otherwise throw during assignment in which case a new object will be created that shares a prototype chain with the original {\ttfamily Error}. The {\ttfamily .stack} and {\ttfamily .message} properties will be carried over from the original error and {\ttfamily .code} or any passed properties will be set on it.\hypertarget{md_backend_2node__modules_2err-code_2_r_e_a_d_m_e_autotoc_md892}{}\doxysection{\texorpdfstring{Tests}{Tests}}\label{md_backend_2node__modules_2err-code_2_r_e_a_d_m_e_autotoc_md892}
