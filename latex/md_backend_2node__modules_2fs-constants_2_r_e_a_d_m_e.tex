 Small module that allows you to get the fs constants across Node and the browser.


\begin{DoxyCode}{0}
\DoxyCodeLine{npm\ install\ fs-\/constants}

\end{DoxyCode}


Previously you would use `require(\textquotesingle{}constants'){\ttfamily for this in node but that has been deprecated and changed to}require(\textquotesingle{}fs\textquotesingle{}).constants\`{} which does not browserify.

This module uses `require(\textquotesingle{}constants'){\ttfamily in the browser and}require(\textquotesingle{}fs\textquotesingle{}).constants\`{} in node to work around this\hypertarget{md_backend_2node__modules_2fs-constants_2_r_e_a_d_m_e_autotoc_md1316}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md_backend_2node__modules_2fs-constants_2_r_e_a_d_m_e_autotoc_md1316}

\DoxyCodeLine{}
\DoxyCodeLine{console.log('constants:',\ constants)}
