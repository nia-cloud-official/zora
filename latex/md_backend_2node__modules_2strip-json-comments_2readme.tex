 \begin{quote}
Strip comments from JSON. Lets you use comments in your JSON files! \end{quote}


This is now possible\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\{}
\DoxyCodeLine{\ \ \ \ //\ rainbows}
\DoxyCodeLine{\ \ \ \ "{}unicorn"{}:\ /*\ ❤\ */\ "{}cake"{}}
\DoxyCodeLine{\}}

\end{DoxyCode}


It will replace single-\/line comments {\ttfamily //} and multi-\/line comments {\ttfamily /\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}/} with whitespace. This allows JSON error positions to remain as close as possible to the original source.

Also available as a \href{https://github.com/sindresorhus/gulp-strip-json-comments}{\texttt{ gulp}}/\href{https://github.com/sindresorhus/grunt-strip-json-comments}{\texttt{ grunt}}/\href{https://github.com/sindresorhus/broccoli-strip-json-comments}{\texttt{ broccoli}} plugin.\hypertarget{md_backend_2node__modules_2strip-json-comments_2readme_autotoc_md3767}{}\doxysection{\texorpdfstring{Install}{Install}}\label{md_backend_2node__modules_2strip-json-comments_2readme_autotoc_md3767}
