\hypertarget{md_backend_2node__modules_2brace-expansion_2_r_e_a_d_m_e_autotoc_md364}{}\doxysubsection{\texorpdfstring{var expanded = expand(str)}{var expanded = expand(str)}}\label{md_backend_2node__modules_2brace-expansion_2_r_e_a_d_m_e_autotoc_md364}
Return an array of all possible and valid expansions of {\ttfamily str}. If none are found, {\ttfamily \mbox{[}str\mbox{]}} is returned.

Valid expansions are\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{/\string^(.*,)+(.+)?\$/}
\DoxyCodeLine{//\ \{a,b,...\}}

\end{DoxyCode}


A comma separated list of options, like {\ttfamily \{a,b\}} or {\ttfamily \{a,\{b,c\}\}} or {\ttfamily \{,a,\}}.


\begin{DoxyCode}{0}
\DoxyCodeLine{/\string^-\/?\(\backslash\)d+\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+(\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+)?\$/}
\DoxyCodeLine{//\ \{x..y[..incr]\}}

\end{DoxyCode}


A numeric sequence from {\ttfamily x} to {\ttfamily y} inclusive, with optional increment. If {\ttfamily x} or {\ttfamily y} start with a leading {\ttfamily 0}, all the numbers will be padded to have equal length. Negative numbers and backwards iteration work too.


\begin{DoxyCode}{0}
\DoxyCodeLine{/\string^-\/?\(\backslash\)d+\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+(\(\backslash\).\(\backslash\).-\/?\(\backslash\)d+)?\$/}
\DoxyCodeLine{//\ \{x..y[..incr]\}}

\end{DoxyCode}


An alphabetic sequence from {\ttfamily x} to {\ttfamily y} inclusive, with optional increment. {\ttfamily x} and {\ttfamily y} must be exactly one character, and if given, {\ttfamily incr} must be a number.

For compatibility reasons, the string {\ttfamily \$\{} is not eligible for brace expansion.\hypertarget{md_backend_2node__modules_2brace-expansion_2_r_e_a_d_m_e_autotoc_md365}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md_backend_2node__modules_2brace-expansion_2_r_e_a_d_m_e_autotoc_md365}
