Given a string as the argument, it will be transformed according to the following rules and the new string will be returned\+:


\begin{DoxyEnumerate}
\item Split into words separated by space characters ({\ttfamily 0x20}).
\item Upper case the first character of each word.
\item Join the words together with no separator.
\item Remove all non-\/word ({\ttfamily \mbox{[}0-\/9a-\/z\+\_\+\mbox{]}}) characters.
