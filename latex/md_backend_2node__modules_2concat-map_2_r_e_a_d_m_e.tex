Return an array of concatenated elements by calling {\ttfamily fn(x, i)} for each element {\ttfamily x} and each index {\ttfamily i} in the array {\ttfamily xs}.

When {\ttfamily fn(x, i)} returns an array, its result will be concatenated with the result array. If {\ttfamily fn(x, i)} returns anything else, that value will be pushed onto the end of the result array.\hypertarget{md_backend_2node__modules_2concat-map_2_r_e_a_d_m_e_autotoc_md523}{}\doxysection{\texorpdfstring{install}{install}}\label{md_backend_2node__modules_2concat-map_2_r_e_a_d_m_e_autotoc_md523}
