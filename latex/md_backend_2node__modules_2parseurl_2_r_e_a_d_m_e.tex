\hypertarget{md_backend_2node__modules_2parseurl_2_r_e_a_d_m_e_autotoc_md2666}{}\doxysubsection{\texorpdfstring{parseurl(req)}{parseurl(req)}}\label{md_backend_2node__modules_2parseurl_2_r_e_a_d_m_e_autotoc_md2666}
Parse the URL of the given request object (looks at the {\ttfamily req.\+url} property) and return the result. The result is the same as {\ttfamily url.\+parse} in Node.\+js core. Calling this function multiple times on the same {\ttfamily req} where {\ttfamily req.\+url} does not change will return a cached parsed object, rather than parsing again.\hypertarget{md_backend_2node__modules_2parseurl_2_r_e_a_d_m_e_autotoc_md2667}{}\doxysubsection{\texorpdfstring{parseurl.\+original(req)}{parseurl.\+original(req)}}\label{md_backend_2node__modules_2parseurl_2_r_e_a_d_m_e_autotoc_md2667}
Parse the original URL of the given request object and return the result. This works by trying to parse {\ttfamily req.\+original\+Url} if it is a string, otherwise parses {\ttfamily req.\+url}. The result is the same as {\ttfamily url.\+parse} in Node.\+js core.
